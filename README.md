## Les projets

### 📊 [MarketPulse](./01_MarketPulse) — Excel · Power BI · DAX
Un investisseur particulier suit son portefeuille "au feeling" dans un Excel basique, sans visibilité sur sa diversification, son risque, ni sa performance face à un indice. MarketPulse construit un outil de suivi complet (classeur Excel, dashboard Power BI, rapport de synthèse) qui répond à ces trois questions avec des recommandations concrètes.

### 🗄️ [StockDB](./02_StockDB) — SQL · Python · Automatisation
Un investisseur qui suit une trentaine d'actions passe des heures chaque semaine à vérifier, une par une, si certaines sont devenues intéressantes. StockDB automatise ce filtrage via un pipeline complet : base de données relationnelle, script d'extraction Python, requêtes SQL avec fonctions de fenêtrage.

### 🤖 [PricePredictor](./03_PricePredictor) — Statistiques · Machine Learning · Backtesting
Le Machine Learning peut-il vraiment prédire un cours de bourse ? Une évaluation honnête en trois étapes (exploration statistique, modélisation comparée à une baseline naïve, backtesting avec frais de transaction), qui conclut avec rigueur plutôt qu'avec un résultat qui sonne bien.

## Limites générales

Les données utilisées dans ce portfolio sont **simulées** à des fins de démonstration — elles ne représentent aucune donnée financière réelle et ne doivent jamais être interprétées comme telles. Aucune analyse présentée ici ne constitue un conseil en investissement réglementé.
